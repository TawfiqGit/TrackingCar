A. Compte & onboarding
	- Créer un compte 
  - Ajouter un/des conducteurs (principal + secondaire)
  - Scan QrCode
  - Suivi du statut (en validation / refus / validé)

B. Recherche & sélection d’un véhicule
  - Voir les véhicules disponibles sur carte / liste
  - Filtres : dates, catégorie, status
  - Voir la fiche véhicule : photos, équipements, conditions, règles (kilométrage inclus, zones autorisées…)

C. Réservation
  - Choisir dates/heures + lieu (agence / zone / parking)
  - Voir le récapitulatif
  - Confirmer la réservation

E. Check-in (départ) : “prise du véhicule”
  - Ouvrir l’écran “Démarrer la location”
  - Vérifier l’état du véhicule avec checklist guidée :
  - Photos obligatoires (carrosserie, jantes, intérieur, pare-brise)
  - Kilométrage compteur saisi ou récupéré automatiquement
  - Signature digitale de l’état des lieux + horodatage
  - Validation du départ (génère le contrat / annexe)  ￼

G. Accès véhicule (si car-sharing / connecté)
- Déverrouiller / verrouiller via clé digitale (Bluetooth / boîtier)  ￼
- Localiser le véhicule (dernière position connue)   ￼

H. Check-out (restitution)
- Lancer “Restituer le véhicule”
- Refaire photos + checklist retour
- Saisir kilométrage retour / niveau carburant
- Confirmer restitution (signature)
- Voir le calcul final : extras (km sup, carburant manquant, retard, nettoyage, dommages)
- Payer le solde / recevoir remboursement caution
- Noter le service + laisser un commentaire
